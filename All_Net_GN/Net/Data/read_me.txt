The folder contains the original slitting data set. The organizational structure is as follows:

  +train               *training set
    +images            *Raw data in.tif format
      +1-C11             
      +2-C12_real
      +3-C12_imag
      +4-C22
      +5-alpha
      +6-anisotropy
      +7-entropy

    +labels_0-1        *Data tag with 0 and 1 pixel values in. png format
    +labels_0-255      *Visualization of data tags with 0 and 255 pixel values in.png format

 +test                 *test set
    +images            *Raw data in.tif format
      +1-C11             
      +2-C12_real
      +3-C12_imag
      +4-C22
      +5-alpha
      +6-anisotropy
      +7-entropy

    +labels_0-1        *Data tag with 0 and 1 pixel values in. png format
    +labels_0-255      *Visualization of data tags with 0 and 255 pixel values in.png format


 +eval                 *Validation set
    +images            *Raw data in.tif format
      +1-C11             
      +2-C12_real
      +3-C12_imag
      +4-C22
      +5-alpha
      +6-anisotropy
      +7-entropy

    +labels_0-1        *Data tag with 0 and 1 pixel values in. png format
    +labels_0-255      *Visualization of data tags with 0 and 255 pixel values in.png format
